1. 인터셉터와 가드의 차이점

1-1. 역할
인터셉터 : 요청 또는 응답을 가공 (데이터 변환, 로깅, 캐싱 등) 
가드 : 요청을 차단하거나 허용 (인증/권한 관리) 

1-2. 실행 시점
인터셉터 : 컨트롤러 실행 전후 (둘 다 가능)
가드 : 컨트롤러 실행 전

1-3. 주요 용도
인터셉터 : 응답 데이터 변환, 로깅, 캐싱 등
가드 : 인증(Auth), 권한(Permission) 확인	

1-4. 사용 데코레이터
인터셉터 : @UseInterceptors(InterceptorClass)
가드 : @UseGuards(GuardClass)

1-5. 반환값
인터셉터 : 요청 데이터를 변경하거나, 응답을 변환 가능
가드 : true (허용) 또는 false (거부)

1-6. 전역 적용 가능
인터셉터 : 가능
가드 : 가능
