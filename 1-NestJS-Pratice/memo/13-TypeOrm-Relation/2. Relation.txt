1. 관계 설정 코드
=> Entity에서 데코레이터를 통해 관계를 설정
=> 이 후에 Controller에서 report 데이터와 user 데이터를 가져옵니다
=> Service에서 report 엔티티 인스턴스를 생성하여 변수에 반환합니다.
=> 변수에 user를 할당합니다. => 이 과정을 통해 fk가 설정됩니다.
=> 그러나 client에 반환되는 값은 report 데이터 + user 전체 데이터가 반환됩니다.
(password가 노출되어 옳지 않습니다. (암호화됨))


2. dto을 사용해서 원하는 응답만 내보낼 수 있습니다.
=> 컨트롤러의 return이 service의 결과값을 내보냅니다.
=> 이 떄 응답 인터셉터에서 dto와 @Expose(), @Transform()를 통해서 데이터를 조작한 뒤에
 응답을 전송합니다.
=> 원하는 데이터만 내보냅니다.
